Layers of Security

Security
Through Obscurity

When you see a door that has a number pad or even a retina scanner next to it you know that there is something important hiding behind that door. AR brings something to the table which we refer to as “security through obscurity” in this case meaning that by hiding the fact that authentication is required to enter somewhere we gain an extra layer of security.

Code
Scrambling

We want to take one of the oldest known means of constraining physical access, doors, and improve the way we interact with them with the help of AR-Glasses. We additionally wanted to extend the security factor by adding an element of randomness to some parts of the input method so that shoulder surfing could be eliminated.
