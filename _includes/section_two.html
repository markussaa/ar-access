How does it work?

The project resulted in an application built in unity that took advantage of AR-foundation and Manomotions hand tracking gestures. Taking advantage of the camera in a phone made it possible to simulate the experience of AR-glasses where the user could interact with a scrambled “keypad” to enter a password. AR-foundation was used to create and place objects in AR so that they became visible for the user and manomations hand tracking gestures were used to track when the user confirmed a numeric value as input.
